# Walmart Sales Prediction – CSET 369

This project predicts Walmart’s weekly store sales using multiple machine learning and deep learning models.  
Models implemented include Linear Regression, Random Forest, KNN, XGBoost, and an RNN using TensorFlow/Keras.

## Features
- Complete data preprocessing  
- Multiple ML model comparisons  
- RNN-based deep learning architecture  
- Evaluation using R² scores  
- Clean and easy-to-understand workflow  

## How to Run
Upload the notebook to Google Colab and run all cells.

## Technologies
Python, Pandas, NumPy, Scikit-learn, TensorFlow, XGBoost.
